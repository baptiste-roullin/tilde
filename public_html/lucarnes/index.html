Explorateur de webring

ⓘ

Ce petit projet est né de l'envie de montrer un panorama du web indépendant, avec une métaphore de façade d'immeubles de nuit.

Les blogs autorisant l'intégration dans un iframe seraient symboliquement comme des fenêtre éclairées, les autres seraient montrés avec les volets clos.

Mais j'ai été surpris que la grande majorité des blocs fonctionnent.

Tout ça s'est donc transformé en explorateur de webrings, tirés de sources diverses.

(Le code pour détecter les iframes chargés ou non est d'ailleurs tricky et WIP.)
